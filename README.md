# MA CVRPTW Solver

## Introduction

本项目用于解决带时间窗的车辆路径规划问题（CVRPTW）, 基于Nagata Y et al 提出的文化基因算法[1]。其中初始解构造以及车辆数最小化基于[2]。

项目算例使用了Solomon提出的经典测试集，你可以在TestData下找到它。
